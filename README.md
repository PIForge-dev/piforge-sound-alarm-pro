## Key Features

- **Real-time Audio Player**: Instantly plays sirens, beeps, or custom alarm tones when PI tags exceed set limits.
- **Visual Alert Indicators**: Highly visible flashing colors matching the alarm severity.
- **Mute & Snooze**: Control panel for operators to temporarily silence alarms while investigating issues.
- **Custom Threshold Settings**: Set High-High, High, Low, and Low-Low alarm levels with custom colors.
